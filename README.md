# DML_joint_inversion

This GitHub repository contains the Python code used to prepare and evaluate the joint inversion of gravity and magnetic data based on the "variation of information" approach for Dronning Maud Land (DML), East Antarctica. The repository also includes the input data files, the final recovered 3D density and magnetic susceptibility models for Dronning Maud Land, and density and susceptibility measurements from DML rock samples.

A research article related to the 3D joint inversion will soon be submitted to JGR Solid Earth under the title: 

Subglacial 3D crustal architecture of central Dronning Maud Land: Insights from joint gravity and magnetic inversion 

by Maximilian Lowe, Mikhail Ginga, Jörg Ebbing, Antonia Stefanie Ruppel, Nikola Koglin, Andreas Läufer, and Graeme Eagles. A priprint is coming soon.

A preprint will be made available soon.
