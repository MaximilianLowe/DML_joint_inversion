# DML_joint_inversion

This GitHub repository contains the python code to prepare and evaluate “variation of information” joint inversion of gravity and magnetic data in Dronning Maud Land, East Antarctica. This reprository contains also the input data files as well as the final recovered density and susceptibility 3D model for Dronning Maud Land as well as density and susceptibility measurements on DML rock samples. 

A research article related to the 3D joint inversion will be shortly submitted to JGR Solid Earth titled: Subglacial 3D crustal architecture of central Dronning Maud Land: Insights from joint gravity and magnetic inversion by Maximilian Lowe, Mikhail Ginga, Jörg Ebbing, Antonia Stefanie Ruppel, Nikola Koglin, Andreas Läufer, and Graeme Eagles. A priprint is coming soon.
